Design System Analysis

Design System of
TAEJAE AI Graduate School

White-dominant canvas, strategic purple, and sharp geometry. TJ Purple is an accent reserved for heroes, CTAs, badges, and headings — never a background flood. A single Pretendard typeface and right-angle surfaces bind it into one brand.

TJ Purple #522C81 TJ Mint #00A497 Pretendard Variable Sharp Geometry · radius 0 White-dominant

08 — Foundations

Elevation & Depth

Depth is restrained — a 1px border and a white-on-white surface step is preferred over shadow. Color-block sections create their own depth.

Level 0 — Flat

1px border, no shadow. Default cards, inputs, dividers.

Level 1 — Subtle

Border + micro-shadow. Lightly-raised cards, outline buttons.

Level 2 — Floating

Soft layered shadow. Menus, modals, dropdowns, sticky header.

11 — Guidelines

Do's & Don'ts

Do

  • Keep the canvas pure white; place purple only on heroes, CTAs, badges, headings.
  • Use mint for interaction — links, hovers, success, secondary CTAs, eyebrows.
  • Make every surface a right angle (rounded-none) — sharp geometry is the brand.
  • Specify color through tokens; compose layout with Container / Section.
  • Alternate white ↔ pale tint ↔ dark sections for page rhythm.

Don't

  • Don't flood the whole screen with purple — it's an accent, not a background.
  • Don't use rounded corners on content cards or buttons.
  • Don't hardcode raw hex — bg-[#522C81] is forbidden.
  • Don't use the Tailwind default palette — only primary / secondary / warm.
  • Don't invert raw scales in dark mode — only semantic tokens switch.
